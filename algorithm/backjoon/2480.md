# 주사위 세개

```python
dice = list(map(int, input().split()))
res = 0
if dice[0] == dice[1] == dice[2]:
    res = 10000+(dice[0]*1000)
elif dice[0] == dice[1] or dice[0] == dice[2]:
    res = 1000 + (dice[0] * 100)
elif dice[1] == dice[2]:
    res = 1000 + (dice[1] * 100)
else:
    res = max(dice)*100
print(res)
```
