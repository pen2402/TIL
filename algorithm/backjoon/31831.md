# 과민성 대장 증후군

```python
import sys

input = sys.stdin.readline

def solution():
    N, M = map(int, input().split())
    res = current = 0
    for a in map(int, input().split()):
        current += a
        if current < 0:
            current = 0
        elif current >= M:
            res += 1
    print(res)

solution()
```
