# 베르트랑 공준

```python
while 1:
    n = int(input())
    if n == 0:
        break
    p = [True] * (2*n+1)
    p[1] = False
    for i in range(n,2*n+1):
        if p[i]:
            for j in range(2,int(i**0.5)+1):
                if i%j == 0:
                    p[i] = False
                    break
    print(p[n+1:2*n+1].count(True))
```
