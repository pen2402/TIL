# 크리스마스 선물

```python
import heapq

N = int(input())
H = []
for n in range(N):
    A = list(map(int, input().split()))
    if A[0] > 0:
        for i in range(A[0]):
            heapq.heappush(H, -A[i+1])
    elif not A[0]:
        if len(H) > 0:
            print(-heapq.heappop(H))
        else:
            print(-1)
```
