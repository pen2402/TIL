# 숨바꼭질 3

```python
N, K = map(int, input().split())
queue = [[N, 0]]
visited = [0] * 100001
res = 0
visited[N] = 1

while queue:
    X, depth = queue.pop(0)
    if X == K:
        res = depth
        break

    if 100000 >= X*2 >= 0 and not visited[X*2]:
        queue.append([2*X, depth])
        visited[X*2] = 1
    if 100000 >= X-1 >= 0 and not visited[X-1]:
        queue.append([X-1, depth+1])
        visited[X-1] = 1
    if 100000 >= X+1 >= 0 and not visited[X+1]:
        queue.append([X+1, depth+1])
        visited[X+1] = 1

print(res)
```
