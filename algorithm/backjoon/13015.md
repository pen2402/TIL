# 별 찍기 - 23

```python
N = int(input())
star = ['*'*N + ' '*(2*N-3) + '*'*N]
for i in range(1, N-1):
    star += [' '*i + '*' + ' '*(N-2) + '*' + ' '*((2*N-3)-2*i) + '*' + ' '*(N-2) + '*']
star += [' '*(N-1) + '*' + ' '*(N-2) + '*' + ' '*(N-2) + '*']
for i in range(N-2, 0, -1):
    star += [' '*i + '*' + ' '*(N-2) + '*' + ' '*((2*N-3)-2*i) + '*' + ' '*(N-2) + '*']
star += ['*'*N + ' '*(2*N-3) + '*'*N]

print('\n'.join(star))
```
