# 만취한 상범

```python
import sys

input = sys.stdin.readline

def solution():
    is_open = [False]*101
    prefix_sum = [0]*101
    for i in range(1, 11):
        is_open[i**2] = True
    for i in range(1, 101):
        prefix_sum[i] = prefix_sum[i-1]+is_open[i]
    T = int(input())
    for _ in range(T):
        n = int(input())
        print(prefix_sum[n])

solution()
```
