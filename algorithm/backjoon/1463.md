# 1로 만들기

```python
X = int(input())
cnt = 0
while X > 1:
    cnt += 1
    if (X/3)%3 == 0:
        X = X//3
    elif ((X-1)/3)%3 == 0:
        X -= 1
    elif (X/2)%2 == 0:
        X = X//2
    elif (X-1)%3 == 0:
        X -= 1
    elif X%3 == 0:
        X = X//3
    elif X%2 == 0:
        X = X//2
    else:
        X -= 1

print(cnt)
```
