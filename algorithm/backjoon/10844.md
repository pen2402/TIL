# 쉬운 계단 수

```python
def stairs(n):
    if n == 0:
        return [n+1]
    elif n == 9:
        return [n-1]
    else:
        return [n-1, n+1]

li = [i for i in range(1, 10)]
N = int(input())
for i in range(N-1):
    temp = []
    for n in li:
        temp += stairs(n)
    li = temp

print(len(li)%1000000000)
```
