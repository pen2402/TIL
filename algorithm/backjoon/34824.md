# 연대 다음 고대

```python
import sys

input = sys.stdin.readline

def solution():
    N = int(input())
    for _ in range(N):
        name = input().rstrip()
        if name == 'yonsei':
            print('Yonsei Won!')
            return
        elif name == 'korea':
            print('Yonsei Lost...')
            return

solution()
```
