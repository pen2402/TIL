# 이어 붙인 수

```java
class Solution {
    public int solution(int[] num_list) {
        String odd = "";
        String even = "";
        for (int num: num_list) {
            if (num%2 == 0) {
                even += Integer.toString(num);
            }
            else {
                odd += Integer.toString(num);
            }
        }
        return Integer.parseInt(odd)+Integer.parseInt(even);
    }
}
```
